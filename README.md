# My Blog Application

A dynamic blog website built with **Node.js**, **Express.js**, and **EJS**. This application allows users to view blog posts and create new content with support for image uploads.

- **Responsive Design**: Clean and modern UI that looks great on all devices.
- **RESTful Architecture**: Clean routing structure for viewing and creating content.
